# GATK4 Variant Analysis Projects

This repo contains two projects:

1. [**Project 1: Tutorial Reproduction**](project1_tutorial_reproduction/)  
   Reproducing the GATK4 tutorial from kpatel427 (https://github.com/kpatel427).

2. [**Project 2: Snakemake + Spiking**](project2_snakemake_spiking/)  
   Snakemake pipeline for WES sample with spiked variant.

---

## Goals

This demonstrates:
- Proficiency in GATK4 variant calling
- Ability to build reproducible pipelines (Snakemake)
- Validation via spiking known variants
- Commitment to documentation and QC
